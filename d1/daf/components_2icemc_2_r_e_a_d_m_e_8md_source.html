1 # icemc
2 [icemc](https://github.com/anitaNeutrino/icemc) is a tool to simulate neutrino interactions in the ice.
3 
4 ## Prerequisites ##
5 
6 To compile icemc you (should) only need a working [ROOT](https://root.cern.ch/downloading-root) installation.
7 
8 Optional extras include the ANITA libraries [libRootFftwWrapper](https://github.com/nichol77/libRootFftwWrapper) and [eventReaderRoot](https://github.com/anitaNeutrino/eventReaderRoot) libraries.
9 The recommended way to install these is with the [anitaBuildTool](https://github.com/anitaNeutrino/anitaBuildTool) which downloads and compiles the core ANITA software libraries.
10 (See the ANITA-3 notes section for more information.)
11 
12 ## Getting icemc ##
13 
14 ``` bash
15 git clone https://github.com/anitaNeutrino/icemc
16 ```
17 ``` bash
18 make
19 ```
20 Currently the Makefile tries to figure out whether to compile against the ANITA libraries by looking for the `ANITA_UTIL_INSTALL_DIR` environment variable.
21 The Makefile may require some love by uncommenting or commenting out various site specific flags at the top.
22 Makefiles are a dark art so if you need help send an email or message someone in the slack [#anita_simulation](https://anitamission.slack.com/messages/anita_simulation/) or [#github_commits](https://anitamission.slack.com/messages/github_commits/) rooms.

24 ## Running icemc ##
25 
26 To run icemc you need first to define two environment variables and then add them to your (DY)LD_LIBRARY_PATH, so that icemc knows where to look for its input files:
27 
28  * ICEMC_SRC_DIR should point to the directory where all your source code is (i.e. this directory)
29  * ICEMC_BUILD_DIR should point to the directory where your exectutables and .pcm are (in case you run icemc by itself it's again this directory, but in case you installed icemc from the anitaBuildTool, it should be something different)
30 
31 An example is:
32 ```bash
33 export ICEMC_SRC_DIR=/path/to/anitaBuildTool/components/icemc/
34 export ICEMC_BUILD_DIR=/path/to/anitaBuildTool/build/components/icemc/
35 export DYLD_LIBRARY_PATH=${ICEMC_SRC_DIR}:${ICEMC_BUILD_DIR}:${DYLD_LIBRARY_PATH}
36 ```
37 
38 If you use the anitaBuildTool, it's a good practice to add these lines to your setup script.
39 
40 
41 To run icemc do:
42 ``` bash
43 ./icemc -i {inputFile} -o {outputDirectory} -r {runNumber} -n {numberOfNeutrinos} -t {triggerThreshold} -e {energyExponent}
44 ```
45 For only running over a single event, the ```-x {eventNumber}``` option can be used.
46 
47 If parameters are not specified
48 
49  * inputs from inputs.conf are used
50  * the output directory is output
51  * the run number is
52  * number of neutrinos to generate as is input file
53  * trigger threshold for full band as defined in anita.hh for that flight
54 
55 ## ANITA formatted data output ##
56 
57 To produce Anita-like output files, you must have the [libRootFftwWrapper](https://github.com/nichol77/libRootFftwWrapper) and [eventReaderRoot](https://github.com/anitaNeutrino/eventReaderRoot) libraries installed before compiling icemc.
58 
59 After that you will need to find and uncomment the site specific flag `ANITA3_EVENTREADER=1` in the Makefile.
60 The next time you compile and run icemc, the ANITA-like outputs will be produced automatically.
61 The outputs also include the TruthAnitaEvent tree with the Monte Carlo truth information about the simulated events.

63 ## ANITA-3 notes ##
64 
65  * If you want to generate anita-3 simulations with the impulse response and noise you should turn on the flags in the input file as by default they are off
66 
67 * If you want to generate a pure thermal noise sample you should to turn on these flags:
68  * Set signal to 0 to measure noise hits
69  * skip cuts on neutrinos that won’t reach the payload (this is just to have it running quicker as we 0’ed the signal anyway)
70  * Min bias flag (so that everything gets saved)
