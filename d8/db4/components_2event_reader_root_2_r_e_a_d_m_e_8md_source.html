9 Introduction
10 ============
11 This is version 4 of the ANITA Event Reader for ROOT Data and Raw Data, and is the first designed for ANITA II data. At present backwards compatibility is broken this will hoepfully be restored at some point once I work out how to do it. It is meant to be a simple class that will read in the raw event data (in ROOT file or raw psev_ format ) and convert it into UsefulAnitaEvent calibrated objects (the raw data reader actually creates text files of times and voltages).

22 Installation
23 ============
24 Checkout the code from the SVN repository, eg.:
25 svn co https://delos.mps.ohio-state.edu/anitaGround/eventReaderRoot/trunk myEventReaderDir
26 Define the ANITA_UTIL_INSTALL_DIR to point to the location you want the library installed (the library files will end up in (ANITA_UTIL_INSTALL_DIR)/lib and the header files in (ANITA_UTIL_INSTALL_DIR)/include).
27 Do
28 make
29 make install

35 Troubleshooting
36 ================
37 There are a myriad of ways that one can run into problems with the event reader, the most common of which are listed here
38 Path problems -- the bane of poorly organised code. By far the easiest way to use the ANITA offline code is to just set the ANITA_UTIL_INSTALL_DIR and have it point to the location you want to install all the packages. If things are set up correctly you will end up with a ANITA_UTIL_INSTALL_DIR/include and ANITA_UTIL_INSTALL_DIR/lib and ANITA_UTIL_INSTALL_DIR/share/anitaCalib all populated with essential headers, libraries and calibration constants. A quick round of
39 make clean all install
40 in libRootFftwWrapper and eventReader (and magicDisplay, etc.) can solve most such difficulties.
41 Calibration data -- AnitaEventCalibrator looks for calibration data in the following order (make sure it finds some).
42 ANITA_CALIB_DIR
43 ANITA_UTIL_INSTALL_DIR/share/anitaCalib
44 ./calib

51 In AnitaGeomTool and therefore in the rest of the code it should be noted that a 'C' style numbering convention is used. Therefore antennas are numbered 0-31 and phi sectors numbered 0-15.

53 As of version 3.2.0 AnitaGeomTool has an interface to Kurt's photogrammetry data.In particular through the following family of functions:
54 a) Double_t getAntZ(int ant); // along with x,y,z,r and phi
55 b) Double_t getAntFaceZ(int ant) // along with x,y,z,r and phi
56 The positions of the centre of the antenna face (the numbers from Kurt) are given by the b) family of functions. The a) family of functions return a position 20cm behind the face of the antenna, this position is meant to approximate teh timing centre of the antenna. At the moment these values are from the ANITA-I flighgt.
