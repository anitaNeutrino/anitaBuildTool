21 ## Building for the first time
22 
23 In theory all one needs to do is make sure the ANITA_UTIL_INSTALL_DIR enviromental vairable is set and then type:
24 
25  ./buildAnita.sh
26 
27 The build script is not very sophisticated it will attempt to:
28 
29  1) Checkout the libraries from GitHub
30  2) Make a build directory and run cmake in that directory to generate Makefiles
31  3) Build the libraries (make)
32  4) Install the libraries in to ANITA_UTIL_INSTALL_DIR (make install)
33 
34 
35 This will build and install everything with the defaults.
36 However, there are some optional arguments
37 
38  ./buildAnita.sh [njobs = 1] [configure = 0] [build aware = 0] [nuke build dir = 1] [convolve tuff responses = 1]
39 
40 | Argument | Default | What it does |
41 | ------------- | --- | ------------- |
42 | njobs | 1 | The number of threads you wish to compile with. njobs > 1 will result in faster compilation, however, warnings and errors may appear out of order. |
43 | configure | 0 | Edit compiler flags and other options before building. |
44 | build aware | 0 | Whether or not to download and build AWARE in addition to the default packages. |
45 | nuke build dir | 1 | Whether or not the build directory will be deleted and completely re-compiled. |
46 | convolve tuff responses | 1 | Whether or not you want to convolve together the TUFF models with the A3 and A4 impulse responses |
47 
48 The main prerequisites are:
49 ROOT https://root.cern.ch with Minuit2, MathMore, and Fortran. See https://root.cern.ch/build-prerequisites to make sure you have the prerequisites required for those features.
50 FFTW http://www.fftw.org
51 cmake https://cmake.org
52 these should be available using your favourite package manager for your system.

55 ## Building for the Nth time
56 
57 After buildAnita.sh is run once, more things are possible.
58 anitaBuildTool includes a high level Makefile, which allows for separately compiling/updating/installing the software, rather than doing all of the above.
59 The Makefile commands are documented, to see what is available do
60 
61  make help
62 
63 in the anitaBuildTool directory.

65 ## clone_method
66 
67 Edit the top line of the clone method file (from https to ssh) to clone over ssh rather than https.
68 This requires uploading your public ssh-key to GitHub.
69 
70 ## If you want to update all the components, you can run:
71 
72  ./updateComponents.sh [Nuke build directory?] [update aware?] [update treeMaker?]
73 
74 If you put a zero for [Nuke build directory?] it will possibly allow you to quicky rebuild only things that have been changed.
75 It might not work though if there are significant modifications.
