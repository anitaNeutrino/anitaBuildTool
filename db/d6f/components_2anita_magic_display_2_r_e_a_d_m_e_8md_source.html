6 ## Introduction
7 
8 Okay so it's a somewhat stupid name, but what can you do I only have some many hours in the day to come up with names.
9 
10 Q. What is it?
11 
12 A. It is an event display and plotting program for use with ANITA-III/IV data in ROOT format.
13 
14 Q. Why should I use it?
15 
16 A. That's a very good question, maybe you shouldn't. I find it is a useful way to look at the rootified data. I know there a people who don't like the rootified data and that's fine an updated AnitaViewer will do everything this does and a whole lot more.
17 
18 Q. So what's the point?
19 
20 A. My preferred way to look at the data is using the ROOT trees rather than the binary files. I find it easier to implement calibrations and unpacking in one place rather than everyone having to do it themselves. This provides people, well right now me, with a (hopefully) really simple way to look at the ROOT data files.

26 ## Prerequisites and installation
27 
28 There are, unfortunately, several pre requisites for installing and running Magic Display.
29 
30 i) ROOT -- If you don't have it installed get the latest version from http://root.cern.ch
31 
32 ii) FFTW3 -- The Fastest Fourier Transform in the west. If you don't have libfftw3.so on your machine then you need to install it. Most linux distributions have a package ready to install, otherwise head to http://www.fftw.org/
33 
34 iii) Pretty much all the ANITA libraries (eventReaderRoot, libRootFftwWrapper, AnitaAnalysisFramework, AnitaAnalysisTools, UCorrelator...). The recommended way to install MagicDisplay is by using the anitaBuildTool
35 ```bash
36 git clone https://github.com/anitaNeutrino/anitaBuildTool.git
37 ```
38 
39 ## Running
40 
41 i) Get some ANITA III/IV data. UCL has a web facing copy of the ANITA-3 data here: http://www.hep.ucl.ac.uk/uhen/anita/private/anita3/flight1415/root/
42 (The usual ANITA username and password are required)
43 
44 ii) Navigate to the directory where anitaMagicDisplay is installed and edit the runMagicDisplay.C macro to point to where you put the ANITA data
45 
46 iii) Then, to look at run 352, do
47 ```bash
48 root macros/runMagicDisplay.C\(352\)
49 ```
